# 🚀 Quick Start: AI Auto-Substitute System

## In 3 Steps...

### **Step 1: Set Your API Key**
- Open EduAI Pro
- Banner appears: **🔑 Enter Anthropic API Key**
- Get free key: https://console.anthropic.com/keys
- Paste key → Click **✓ Save Key**
- ✅ AI is now active

### **Step 2: Go to Substitutes**
- Click **🔄 Substitutes** in left menu
- You'll see: **"Auto-Assignment Active"** (green) + feature explanation

### **Step 3: Report an Absence**
- Click **📋 Report New Absence**
- Select teacher, subject, class, period, date
- Click **"📋 Report Absence (AI will auto-assign)"**
- **BOOM!** 🤖 AI finds & assigns the best substitute in ~2 seconds

## 🎛️ Two Work Modes

### **AUTO MODE** ✅ (Recommended)
- Absence reported → AI instantly assigns
- Leave approved → AI finds cover automatically
- Zero manual work needed
- Perfect for busy admins

**Enable**: Settings → "Auto AI Suggest" → ON

### **MANUAL MODE** 🎯 (Review & Approve)
- Absence reported → AI suggests 3-5 options
- Admin reviews recommendations
- Admin clicks "Assign [Name]" to confirm
- More control, slightly more work

**Switch**: Settings → "Auto AI Suggest" → OFF

## 🆘 Troubleshooting

❌ **Auto-Assign Not Working?**
- Check: Is API key set? (Settings → check API Key field)
- Check: Is "Auto AI Suggest" toggle ON? (Settings → scroll down)
- Check: Do we have available teachers? (check Availability)

❌ **Wrong Teacher Assigned?**
- Go to: 🔄 Substitutes
- Click: "🤖 AI Suggest" again (get new options)
- Click: "✅ Assign [Different Name]" (override)

❌ **No Substitutes Available?**
- All teachers are busy/on leave
- AI shows: "No available substitutes found"
- Options: Use external substitute or reschedule class
