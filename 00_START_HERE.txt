## 🆘 Common Questions

**Q: Is AI always making decisions?**  
A: Only when "Auto AI Suggest" is ON (Settings). Can toggle OFF for manual mode.

**Q: What if AI picks wrong teacher?**  
A: Go to Substitutes → Click "AI Suggest" → Choose different teacher. Easy override.

**Q: Does it work without API key?**  
A: System still works, but auto-assign won't run. Manual "AI Suggest" shows error.

**Q: How do I trust the AI?**  
A: Claude is trained on education data. Decisions use logical criteria (subject, workload, experience).

**Q: Can I see why AI chose someone?**  
A: Yes! Each assignment shows AI's reasoning in notifications.

**Q: What if all teachers are busy?**  
A: AI announces "No available substitutes" → Use external substitute option.
